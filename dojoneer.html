Dojoneer    Created with Aaron Nguyen and Evan Hyett

A study dojo for users to maintain productivity! It includes a notepad, to do list that is stored in local storage, a task-timer, and a lofi radio that is embed via Youtube. It features a variety of React components such as reactPlayer, react-draggable and others. All components are fully interactive and can be placed wherever the user wishes to do so.
